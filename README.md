# Projeto China Transportes (App de Gerenciamento)

## Descrição do Projeto

Este é um aplicativo Android para gerenciamento de transporte escolar. Ele permite o cadastro e a gestão completa de alunos, responsáveis, tripulantes (motoristas e monitores), escolas e turmas, garantindo organização e fácil acesso às informações.

## Como Começar

O aplicativo já vem com dados de exemplo para demonstração. Para acessar, use as seguintes credenciais ao iniciar o app:

- **E-mail:** `admin@admin.com`
- **Senha:** `admin`

## Funcionalidades

- **Sistema de Autenticação**: Login e Registro de usuários para acesso seguro.
- **Gerenciamento Completo (CRUD)**: Funcionalidades de Criar, Ler, Editar e Deletar para:
  - Alunos
  - Responsáveis
  - Tripulantes
  - Escolas
  - Turmas
- **Interface Intuitiva**: Navegação via menu lateral, layout responsivo e feedback visual para o usuário.
- **Contato Rápido**: Botão de ação flutuante para iniciar uma conversa de suporte via WhatsApp.

## Tecnologias Utilizadas

- **Linguagem**: Kotlin
- **Arquitetura**: MVVM (ViewModel, Repository, LiveData, Flow)
- **Banco de Dados**: Room Persistence Library para armazenamento local.
- **Navegação**: Android Navigation Component para gerenciar o fluxo de telas.
- **Interface**: Material Design Components, RecyclerView, ConstraintLayout, e View Binding para acesso seguro às views.
- **Controle de Versão**: Git e GitHub.
